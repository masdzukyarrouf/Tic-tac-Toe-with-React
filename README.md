# Tic-Tac-Toe with React

A simple Tic-Tac-Toe game built using React and TypeScript  to demonstrate UI creation, state management and component structure.

## Live Demo

Check it out : [Tic Tac Toe](https://tic-tac-toe-withreact.vercel.app/)

## Features
- Two-player mode (X vs O)  
- Highlights winning line  
- Restart / New game button  
- Go back to specified move
- Clean and responsive layout with Tailwind CSS 
- Uses React Hooks for state management

## Tech Stack
- React  
- JavaScript / TypeScript  
- Tailwind CSS   
- Vercel   

## What I learned
- Managing component state with React Hooks (useState, useEffect)
- Building reusable components (Board, Square)
- Styling with Tailwind CSS
- Deploying a React app to  Vercel
